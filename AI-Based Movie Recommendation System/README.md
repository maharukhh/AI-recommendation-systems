# AI-Based Movie Recommendation System

A simple Python-based movie recommendation system that suggests movies based on user-selected categories. The system uses a predefined dataset and returns recommendations instantly based on user input.

## About This Project

This project is a beginner-level recommendation system built using Python. It demonstrates how basic logic and data structures can be used to create a simple recommendation engine without machine learning.

The system recommends movies based on categories such as action, comedy, horror, and sci-fi.

## Features

* Category-based movie recommendations
* Simple and interactive command-line interface
* Instant results based on user input
* Easy to understand and modify
* Beginner-friendly project

## Available Categories

* Action
* Comedy
* Horror
* Sci-Fi

## Technologies Used

* Python
* Basic Data Structures (Dictionary)
* Conditional Statements
* Loops
* User Input Handling

## How It Works

1. The user runs the program.
2. The system displays available categories.
3. User enters a category name.
4. Program checks the dictionary.
5. If category exists, movies are displayed.
6. If category does not exist, an error message is shown.

## Output

The program prints recommendations directly in the terminal based on user input. Screenshots of sample outputs are stored in the `output/` folder.
