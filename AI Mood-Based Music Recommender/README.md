# AI Mood-Based Music Recommender

A simple Python-based music recommendation system that suggests songs according to the user's current mood.

## About This Project

This project recommends songs based on different moods entered by the user. It uses a predefined collection of songs and randomly selects a song that matches the selected mood.

The project demonstrates the basic concepts of:

* Python Dictionaries
* Functions
* Conditional Statements
* Loops
* Random Module
* Simple Recommendation Systems

## Features

* Mood-based song recommendations
* Interactive command-line interface
* Random song selection
* Multiple mood categories
* Easy to customize and expand

## Supported Moods

* Happy
* Sad
* Angry
* Relaxed
* Romantic

## How It Works

1. The user enters a mood.
2. The program searches for the mood in the music database.
3. A random song is selected from the corresponding category.
4. The recommended song is displayed to the user.
