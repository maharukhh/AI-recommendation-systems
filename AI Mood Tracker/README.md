# AI Mood Tracker

A Python-based mood tracking application that helps users record their daily emotions, monitor stress and sleep patterns, and receive simple AI-generated wellness suggestions.

## About This Project

This project allows users to log their moods, write notes about their day, and track factors such as stress levels and sleep hours. The application analyzes mood entries and provides insights to help users better understand their emotional patterns.

The project demonstrates concepts such as:

* Mood Tracking
* Sentiment Analysis
* Data Storage with JSON
* Pattern Analysis
* Basic AI Recommendations
* Python Object-Oriented Programming

## Features

* Daily mood logging
* Personal notes and journaling
* Stress level tracking
* Sleep hour tracking
* Basic sentiment analysis
* Mood summary and statistics
* AI-generated wellness suggestions
* Local data storage using JSON

## How It Works

1. The user records a mood entry.
2. The application saves the data locally in a JSON file.
3. Sentiment analysis is performed on the user's note.
4. The system generates wellness suggestions.
5. A summary of mood patterns is displayed.
